### How to Run the Bot
1.  **Install Python** and `pip`.
2.  **Install Dependencies**:
    ```bash
    pip install python-telegram-bot
    ```
3.  **Configure Token**:
    -   Open `bot.py` and replace `YOUR_TOKEN_HERE` with your actual Bot Token from @BotFather.
4.  **Run**:
    ```bash
    python bot.py
    ```
